## Instrucciones:
+ Clonar repositorio desde GitHub.
+ Habilitar las variables de entorno en un archivo *.env* desde la plantilla *.config.env*.
+ Ejecutar ``npm install`` para instalar las dependencias.
+ ``npm start`` para iniciar.
